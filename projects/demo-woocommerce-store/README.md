# Demo WooCommerce Store

This demo tests the `woocommerce-mini-store` starter.

## Fictional Store

KeyCraft Supply sells mechanical keyboard accessories for small desk setups.

## Goal

Create a small WooCommerce catalog that can test product structure, categories, product SEO, cart, checkout, policies, and launch readiness.

## Demo Outputs

- `product-catalog.md`
- `store-setup.md`

## Implementation Paths

Block theme path:

- Use WordPress + WooCommerce as the actual commerce system.
- Use the product catalog in WordPress admin.
- Use WooCommerce cart, checkout, account, and email flows.

Headless path:

- Use the Next.js `/shop` route as a catalog preview.
- Treat checkout as future integration work unless WooCommerce Store API is added.
