## FAQ

Question: Can I edit the site myself?

Answer: Yes. The site is built around standard WordPress editing so pages and blog posts can be updated from the admin panel.

Question: Is WooCommerce included?

Answer: The first package prepares the site for commerce. A mini-store can be added as a separate WooCommerce phase.

Question: How long does a basic launch take?

Answer: A focused small business site can usually be planned and built faster than a custom frontend application when content is ready.
