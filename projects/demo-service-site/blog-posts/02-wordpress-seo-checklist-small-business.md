## Intro

Small business SEO starts with a clean site structure, clear page intent, useful content, and basic technical setup.

This checklist focuses on the essentials that should be in place before launch.

## Site Structure

Check that the site has:

- homepage
- services page
- about page
- blog
- contact page
- privacy policy

Internal link:

- Link commercial content to `/services/`.

## Page-Level SEO

For each important page:

- choose one primary keyword
- write a clear SEO title
- write a useful meta description
- use a short readable URL
- use one H1
- organize content with H2 and H3 headings

## Blog SEO

For blog content:

- group posts into categories
- create topic clusters
- link supporting posts to pillar pages
- link commercial-intent posts to services
- include a CTA near the end

Internal links:

- Link blog readers to `/contact/` when they are ready to plan a project.
- Link category browsing to `/categories/`.

## Technical SEO

Before launch:

- enable sitemap
- check robots.txt
- connect Analytics
- connect Search Console
- optimize images
- test mobile layout
- check page speed

## FAQ

### Do small businesses need a blog?

Not always, but a blog helps when customers search for questions, comparisons, checklists, and local service advice.

### Is an SEO plugin enough?

No. An SEO plugin helps with metadata and sitemap setup, but the content structure and internal links still need planning.
