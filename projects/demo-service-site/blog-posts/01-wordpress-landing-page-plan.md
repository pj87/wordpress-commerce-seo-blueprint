## Intro

A local business landing page should make the offer clear, build trust quickly, and guide visitors toward one action.

Before opening the WordPress editor, plan the page structure, content blocks, search intent, and conversion path. This prevents the page from becoming a collection of disconnected sections.

## Recommended Structure

Use this order:

1. Hero
2. Problem
3. Solution
4. Benefits
5. Features
6. Social proof
7. Process
8. FAQ
9. Final CTA

## Hero

The hero should answer:

- What do you offer?
- Who is it for?
- What should the visitor do next?

Example CTA:

```text
Book a consultation
```

Internal link:

- Link the CTA to `/contact/`.

## Problem And Solution

Use the problem section to name the visitor's pain:

- slow website
- unclear offer
- poor mobile layout
- weak SEO
- no reliable contact path

Then explain the solution in plain business terms.

Internal link:

- Link solution details to `/services/`.

## SEO Planning

Before publishing:

- choose one primary keyword
- write the SEO title
- write the meta description
- use one H1
- organize sections with H2 headings
- add internal links
- add image alt text

## FAQ

### How long should a landing page be?

Long enough to explain the offer, answer objections, and support the CTA. Avoid adding sections that do not help the visitor decide.

### Should the landing page link to the blog?

Yes, if the blog supports the buying decision. Link to useful guides from the landing page and link back from articles to the service page.

## Final CTA

If you want a landing page structure that connects content, SEO, and conversion, start with the service scope:

```text
/services/
```
