# Theme Installation

## Goal

Install the demo block theme in WordPress.

## Theme Source

Use:

```text
implementation/wordpress-block-theme/
```

## Installation Steps

1. Copy `implementation/wordpress-block-theme/`.
2. Paste it into the WordPress installation:

```text
wp-content/themes/wp-growth-studio/
```

3. Open WordPress admin.
4. Go to Appearance > Themes.
5. Activate `WP Growth Studio`.
6. Open Appearance > Editor.
7. Confirm templates and patterns are visible.

## Verification

- Theme appears in Appearance > Themes.
- Front page uses the demo homepage sections.
- Header and footer render.
- Patterns are available in the block editor.
