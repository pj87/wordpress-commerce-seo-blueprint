# Launch Preparation

## Goal

Prepare the demo site for a public launch.

## Source Checklist

Use:

- `projects/demo-service-site/05-launch-readiness.md`
- `checklists/launch-checklist.md`

## Steps

1. Connect domain.
2. Enable SSL.
3. Confirm homepage and blog page settings.
4. Configure permalinks.
5. Test navigation and footer links.
6. Test the contact form.
7. Configure SEO titles and meta descriptions.
8. Enable sitemap.
9. Install analytics.
10. Connect Search Console.
11. Enable caching.
12. Optimize images.
13. Configure backups.
14. Review admin users.
15. Remove unused plugins.
16. Confirm deployment-specific setup in `implementation/setup/07-deployment.md`.

## Verification

- Public domain loads over HTTPS.
- Contact form submits and sends email.
- Sitemap is available.
- Analytics is active.
- Search Console is connected.
- Backups are configured.
- Mobile layout is reviewed.
- Sitemap and robots routes are verified.
