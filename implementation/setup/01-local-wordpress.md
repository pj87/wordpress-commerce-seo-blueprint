# Local WordPress Setup

## Goal

Create a local WordPress installation where the demo block theme can be tested.

## Options

Use one of:

- Local
- XAMPP
- Docker-based WordPress
- A temporary staging site from hosting

## Required Environment

- WordPress 6.4 or newer
- PHP 8.0 or newer
- MySQL or MariaDB
- Admin access

## Setup Steps

1. Create a new WordPress site.
2. Set the site title to `WP Growth Studio`.
3. Set permalinks to `Post name`.
4. Create these pages:
   - Home
   - Services
   - About
   - Blog
   - Contact
   - Privacy Policy
5. Set `Home` as the homepage.
6. Set `Blog` as the posts page.
7. Create blog categories:
   - WordPress
   - SEO
   - WooCommerce
   - Performance

## Verification

- The admin panel loads.
- Permalinks use readable slugs.
- Homepage and blog page are assigned.
- Categories exist.
