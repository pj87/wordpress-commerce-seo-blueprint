# Workflows

These workflows describe how to run a WordPress project from idea to launch.

Use them in this order for most projects:

1. `01-discovery.md`
2. `02-sitemap-and-content.md`
3. `03-landing-page-build.md`
4. `04-seo-blog-build.md`
5. `05-woocommerce-store-build.md` if commerce is needed
6. `06-launch.md`
7. `07-maintenance.md`

Each workflow defines inputs, steps, outputs, and acceptance criteria.
