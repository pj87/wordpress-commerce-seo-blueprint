# SEO Blog Build Workflow

## Goal

Create a blog structure that supports search traffic, internal linking, and conversion.

## Inputs

- Site niche
- Target audience
- Blog categories
- Keyword ideas
- Service or product pages
- Lead magnet or CTA

## Steps

1. Create the blog index.
2. Create categories around real topic clusters.
3. Define one pillar topic.
4. Plan supporting articles.
5. Write the first 3 articles.
6. Add internal links between pillar, articles, and commercial pages.
7. Add SEO titles and meta descriptions.
8. Optimize images.
9. Add article CTA blocks.
10. Submit or verify sitemap.

## Outputs

- Blog index
- Category pages
- Topic cluster plan
- 3 published posts
- Internal links
- Blog CTA

## Acceptance Criteria

- Each article targets one clear search intent.
- Posts link to related posts and relevant commercial pages.
- Blog categories are useful to visitors.
- Metadata is complete.
- Sitemap includes posts and categories.
