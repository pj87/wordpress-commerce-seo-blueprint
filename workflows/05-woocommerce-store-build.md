# WooCommerce Store Build Workflow

## Goal

Build a small WooCommerce store with a reliable product, cart, checkout, and order flow.

## Inputs

- Product list
- Product categories
- Product images
- Prices
- Shipping rules
- Tax requirements
- Payment gateway
- Store policies

## Steps

1. Install and configure WooCommerce.
2. Set currency, address, tax, and shipping settings.
3. Configure payment methods.
4. Create product categories.
5. Add simple products.
6. Add variable products if needed.
7. Add product images and alt text.
8. Write product descriptions and metadata.
9. Configure cart, checkout, account, and order pages.
10. Add refund, privacy, terms, and shipping policies.
11. Test checkout with a test payment.
12. Confirm order emails for customer and admin.
13. Run a performance and mobile check.

## Outputs

- Working store
- Product catalog
- Cart and checkout
- Order email flow
- Store policies
- Basic product SEO

## Acceptance Criteria

- A customer can place a test order.
- Admin receives the order.
- Customer receives order email.
- Taxes, shipping, and totals are understandable.
- Products have images, categories, descriptions, and metadata.
