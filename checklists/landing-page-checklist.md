# Landing Page Checklist

## Structure

- [ ] Hero section
- [ ] Problem statement
- [ ] Solution section
- [ ] Benefits
- [ ] Features
- [ ] Social proof
- [ ] Process
- [ ] Offer
- [ ] FAQ
- [ ] Final CTA

## Hero

- [ ] Clear headline
- [ ] Specific subheadline
- [ ] Primary CTA
- [ ] Secondary CTA if useful
- [ ] Trust signal
- [ ] Relevant image, mockup, or visual

## Conversion

- [ ] CTA repeated at logical points
- [ ] Contact form tested
- [ ] Form success message configured
- [ ] Mobile layout checked
- [ ] Page speed checked
- [ ] Analytics event configured if needed

## SEO

- [ ] One primary keyword
- [ ] SEO title
- [ ] Meta description
- [ ] Clean URL slug
- [ ] H1 used once
- [ ] Logical H2/H3 structure
- [ ] Image alt text
- [ ] Internal links
