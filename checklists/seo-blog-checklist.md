# SEO Blog Checklist

## Blog Architecture

- [ ] Blog index
- [ ] Category pages
- [ ] Individual article template
- [ ] Author page if needed
- [ ] Related posts
- [ ] Internal linking strategy
- [ ] Newsletter or lead CTA
- [ ] Sitemap enabled

## Article Checklist

- [ ] One clear primary keyword
- [ ] Search intent matched
- [ ] Short readable URL
- [ ] Compelling title
- [ ] Meta description
- [ ] H1 used once
- [ ] Logical H2/H3 structure
- [ ] Images optimized
- [ ] Image alt text
- [ ] Internal links
- [ ] External references where useful
- [ ] FAQ section if useful
- [ ] CTA included
- [ ] Updated date visible if relevant

## Topic Cluster

- [ ] Pillar page identified
- [ ] Supporting articles planned
- [ ] Internal links back to pillar
- [ ] Related commercial page linked
