# WordPress Launch Checklist

## Core Site

- [ ] Domain connected
- [ ] SSL active
- [ ] Permalinks configured
- [ ] Homepage set
- [ ] Blog page set
- [ ] Navigation menu configured
- [ ] Footer links configured
- [ ] 404 page checked
- [ ] Privacy policy added
- [ ] Cookie notice added if needed

## Forms And Tracking

- [ ] Contact form tested
- [ ] Email delivery tested
- [ ] Analytics installed
- [ ] Search Console connected
- [ ] Sitemap enabled
- [ ] Robots.txt checked

## SEO

- [ ] SEO plugin configured
- [ ] Titles set
- [ ] Meta descriptions set
- [ ] Open Graph preview checked
- [ ] Key pages indexed or submitted

## Performance And Security

- [ ] Cache enabled
- [ ] Images optimized
- [ ] Mobile layout tested
- [ ] Backup configured
- [ ] Security basics configured
- [ ] Unused plugins removed
- [ ] Admin users reviewed
- [ ] Updates applied
