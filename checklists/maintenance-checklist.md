# Maintenance Checklist

## Weekly

- [ ] Check updates
- [ ] Check backups
- [ ] Check uptime
- [ ] Check contact forms
- [ ] Review security alerts

## Monthly

- [ ] Update plugins and themes carefully
- [ ] Test checkout if WooCommerce is active
- [ ] Review analytics
- [ ] Review Search Console
- [ ] Optimize new images
- [ ] Clean spam comments
- [ ] Check broken links

## Before Major Updates

- [ ] Create backup
- [ ] Test on staging if possible
- [ ] Update plugins
- [ ] Update theme
- [ ] Update WordPress core
- [ ] Test critical flows
