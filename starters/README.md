# Starter Kits

Starter kits define project shape before implementation.

Use a starter kit when you want to build a specific project type quickly:

- `local-business-site.md`
- `service-landing-blog.md`
- `woocommerce-mini-store.md`
- `seo-affiliate-blog.md`
- `headless-wordpress-blog.md`

Each starter includes the project goal, pages, content model, plugins, build order, and done criteria.
