# Service Landing Page + Blog Starter

## Goal

Build a client-style service website with a conversion landing page and SEO blog.

## Example

WordPress landing page service for local businesses.

## Pages

- Home
- Services
- About
- Blog
- Contact
- Privacy policy

## Blog Categories

- WordPress
- SEO
- WooCommerce
- Performance

## Required Features

- Responsive design
- Contact form
- SEO metadata
- Blog index
- Category pages
- 3 blog posts
- CTA sections
- FAQ section
- Analytics
- Sitemap

## Build Order

1. Complete discovery.
2. Draft sitemap and page goals.
3. Build homepage as the primary landing page.
4. Build services, about, and contact pages.
5. Configure blog categories.
6. Publish 3 articles.
7. Add internal links from posts to service pages.
8. Configure SEO metadata, sitemap, analytics, and forms.
9. Run launch checklist.

## Done Criteria

- Main landing page has hero, benefits, process, FAQ, and final CTA.
- Blog has 3 useful posts and clear categories.
- Contact form works.
- Service CTA appears on relevant blog posts.
- SEO basics are configured.
