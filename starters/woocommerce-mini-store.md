# WooCommerce Mini-Store Starter

## Goal

Build a small WooCommerce store that proves the full purchase flow.

## Example

Mechanical keyboard accessories store.

## Pages

- Home
- Shop
- Product detail
- Cart
- Checkout
- My account
- Order confirmation
- Contact
- Terms
- Privacy
- Refund policy

## Product Scope

- 5 simple products
- 2 variable products
- 3 product categories
- Product images
- Product descriptions
- Product SEO metadata

## Recommended Plugins

- WooCommerce
- SEO plugin
- Cache/performance plugin
- Backup plugin
- Security plugin
- Analytics integration

## Build Order

1. Configure WooCommerce basics.
2. Add store pages.
3. Add categories.
4. Add simple products.
5. Add variable products.
6. Configure payments, shipping, and taxes.
7. Add store policies.
8. Test cart, checkout, account, and emails.
9. Optimize product and category metadata.
10. Run launch checklist.

## Done Criteria

- Test customer can place an order.
- Customer and admin emails are received.
- Products have images, descriptions, categories, and metadata.
- Checkout totals are understandable.
- Policies are linked from footer and checkout.
