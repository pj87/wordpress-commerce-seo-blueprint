# Learning Path

The fastest practical path is:

1. WordPress basics
2. Landing pages
3. SEO blog structure
4. Themes and site editing
5. Essential plugins
6. WooCommerce store setup
7. Store SEO
8. Performance
9. Security
10. Deployment and maintenance
11. Optional headless WordPress

## Six-Week Plan

### Week 1: WordPress Basics

- Install WordPress locally or on hosting.
- Learn posts, pages, categories, tags, media, menus, widgets, users, and settings.
- Build a simple five-page website.

### Week 2: Landing Pages

- Build a conversion-focused landing page.
- Include hero, problem, solution, benefits, features, process, FAQ, and CTA sections.
- Add a working contact form.

### Week 3: SEO Blog

- Create blog categories.
- Plan a topic cluster.
- Publish three optimized posts.
- Add internal links and basic metadata.

### Week 4: WooCommerce Basics

- Install WooCommerce.
- Configure currency, taxes, shipping, payments, cart, and checkout.
- Add simple and variable products.

### Week 5: Store SEO, Performance, and Security

- Optimize product and category pages.
- Configure caching, image optimization, backups, and security basics.
- Test critical flows.

### Week 6: Deployment and Maintenance

- Launch a public WordPress site or store.
- Connect domain, SSL, analytics, Search Console, sitemap, backups, and update workflow.
