# Project Types

## Project 1: SEO Landing Page + Blog

Goal: build a client-style website for a fictional service business.

Required pages:

- Homepage
- Services
- About
- Blog
- Contact
- Privacy policy

Required features:

- Responsive design
- Contact form
- SEO metadata
- Blog index
- Category pages
- 3 blog posts
- CTA sections
- FAQ section
- Analytics
- Sitemap

## Project 2: WooCommerce Store

Goal: build a small e-commerce store.

Required pages:

- Homepage
- Shop
- Product detail
- Cart
- Checkout
- My account
- Order confirmation
- Contact
- Terms
- Privacy
- Refund policy

Required features:

- 5 simple products
- 2 variable products
- Product categories
- Product images
- Cart and checkout
- Test payment
- Customer account
- Order emails
- Basic SEO
- Performance optimization
- Security basics

## Project 3: Optional Headless WordPress

Goal: use WordPress as a CMS and build the frontend separately.

Suggested stack:

- WordPress
- WordPress REST API
- Next.js
- TypeScript
- React

Required features:

- Fetch posts from WordPress
- Render blog index
- Render post detail
- Fetch pages
- Fetch products if WooCommerce is used
- Custom frontend styling
- Deployment
