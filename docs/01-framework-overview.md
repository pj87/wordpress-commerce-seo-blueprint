# Framework Overview

This framework is designed for fast, commercial WordPress delivery.

It focuses on projects that clients and small businesses commonly need:

- Landing pages
- SEO blogs
- Business websites
- Portfolio sites
- Lead-generation websites
- WooCommerce stores
- Small business e-commerce
- Headless WordPress integrations

## Core Principle

Use WordPress when speed, editing, publishing, SEO, and client ownership matter more than custom application architecture.

## Best Fit

WordPress is a strong fit for:

- CMS editing
- Blogging
- SEO content publishing
- Marketing pages
- Client-editable content
- Plugin-supported features
- WooCommerce
- Fast delivery
- Non-technical admin users

## Poor Fit

Avoid WordPress as the primary platform for:

- Highly custom SaaS products
- Real-time applications
- Complex dashboards
- Advanced custom authorization
- Large custom backends
- Highly interactive app-like interfaces

Use WordPress headlessly only after the classic WordPress workflow is understood.

## Delivery Model

Every project should move through:

1. Discovery
2. Sitemap
3. Wireframe
4. Content collection
5. WordPress setup
6. Theme and page build
7. SEO setup
8. Forms and integrations
9. Performance pass
10. Security and backup setup
11. Review
12. Launch
13. Maintenance
